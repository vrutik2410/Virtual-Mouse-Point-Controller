Virtual Mouse Point Controller

ABOUT THE PROJECT
>> Virtual Mouse Controller is a web bassed application which allows the user to almost have 0 human contact with the mouse board on the computer.
>> The diffrent movements of fingers are recognised by the application and accordingly action are taken. The front-end is coded in Flask which is a python based framework while the backnd is coded in pure python.
>> The Python code have used several libraries which needs to be installed seperatly.The list is given in the folder.


The following is the vedio demo of the gestures for respective command. It must be taken care that the hand must be kept such that the whole of it is in the frame of the camera.



Gesture of cursor movement

Cursor is assigned to the midpoint of index and middle fingertips. This gesture moves the cursor to the desired location. Speed of the cursor movement is proportional to the speed of hand.

Gesture for single left click

The single left click is done when the index finger is bent to 90 degree from the static V-position.

Gesture for single right click

The single right click is done when the middle finger is bent to 90 degree from the static V-position.

Gesture for double click

The middle and index finger are touched together and this position is kept static for more than 1 sec.

Gesture for scroll

Dynamic Gestures for horizontal and vertical scroll. The speed of scroll is proportional to the distance moved by pinch gesture from start point. Vertical and Horizontal scrolls are controlled by vertical and horizontal pinch movements respectively.

Gesture for drag and drop

The drag is done after selecting the the file/folder.The dropping is done when the palm is closed as in the grasping position and the folder is then dragged and ropped to the desired location.

It can be used to move/tranfer files from one directory to other.

Gesture to select multiple items

The palm is closed as in grasping position and moved in the area where the files are to be selected

Gestures for Volume control

The rate of increase/decrease of volume is proportional to the distance moved by pinch gesture from start point.

Gestures for Brightness control

The rate of increase/decrease of brightness is proportional to the distance moved by pinch gesture from start point.
